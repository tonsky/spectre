Installing Spectre¶

  1. Setup Fantom 1.0.58 from official site;
  2. Download and unpack latest Spectre release from spectreframework.org;
  3. Add <SPECTRE_DIR>/bin to your classpath.

Testing installation¶

To test you have everything set up properly, run the following command::

>>> spectre rundevserver <SPECTRE_DIR>/examples/tour_de_force/

And point you browser at http://localhost:8080/. If you see Spectre demo app, than you’ve done everything all right.

Congratulations! You now have Spectre installed. Proceed to the Getting started section.
